# Portfolio Website

This is a Next.js portfolio website designed with Tailwind CSS and Framer Motion.

## Features

- **Interactive Hero Section**: Animated entrance with your hero image.
- **Portfolio Explorer**: Browse work by Industry -> Client -> Project Showcase.
- **Animations**: Smooth transitions and hover effects using Framer Motion.
- **Responsive Design**: Works on desktop and mobile.

## How to Run

1.  Navigate to the site directory:
    ```bash
    cd site
    ```

2.  Install dependencies (if you haven't already):
    ```bash
    npm install
    ```

3.  Start the development server:
    ```bash
    npm run dev
    ```

4.  Open [http://localhost:3000](http://localhost:3000) in your browser.

## Customization

-   **Hero Image**: Replace `public/Hero.png` with your own image if needed.
-   **Data**: Edit `src/components/Portfolio.tsx` to update the list of Industries, Clients, and Projects.
-   **Colors/Fonts**: Edit `src/app/globals.css` and Tailwind classes.
